Model 5-3 重回帰

学生達のある1年間の出席率を - カテゴリ変数A: アルバイトが好きかどうか - 連続値変数Score: 学問への興味の強さ の二変数で線形回帰する, というもの.

モデルを記述したstanファイルとモデルに食わせるデータは所与して, 以下のように出力を得たとする. もしこれらの中身が気になったなら書籍のサポートページに飛ぶべし.

fit <- stan(file = stan_path, data = data, seed = 1234)
ms <- rstan::extract(fit)

Figure 5-2

モデルのあてはまりの良さを可視化するために, 80%信頼区間の上に観測データをプロットする.

bayesplot::ppc_ribbon_groupedに投げる. facet_argsはbayesplot::ppc_ribbon_groupedが呼び出すggplot::facet_wrapに与えるパラメーターをlist形式で受け取る. labellerを以下のように与えないとgroup: 0のように表示されるので, 明示的にas_labellerによって表示形式を指定している.

color_scheme_set("brightblue")
bayesplot::ppc_ribbon_grouped(y = d$Y, yrep = ms$y_pred, x = d$Score, group = d$A, facet_args = list(scales = "fixed", labeller = as_labeller(c("0" = "A=0", "1" = "A=1"))), prob_outer = 0.8, y_draw = "both") +
    panel_bg(fill = "gray90", color = NA) +
    grid_lines(color = "white") +
    labs(x = "Score", y = "Observed and Predicted y", title = "50% and 80% posterior predictive zone of Y vs Score")

Figure 5-3

説明変数が増えてプロットの次元が上がると先の方法による可視化は難しくなる. 説明変数が増えても可能な方法としては, 実測値と予測値の比較, 推定されたノイズの分布確認などがある. 前者は, 呼び出す関数がbayesplot::ppc_intervals_groupedに変わるだけで, やっていることは一つ前と実質同じ.

color_scheme_set("purple")
bayesplot::ppc_intervals_grouped(y = d$Y, yrep = ms$y_pred, x = d$Y, group = d$A, facet_args = list(scales = "fixed", labeller = as_labeller(c("0" = "A=0", "1" = "A=1")))) +
    panel_bg(fill = "gray90", color = NA) +
    grid_lines(color = "white") +
    labs(x = "Observed y", y = "Predicted y", title = "50% and 80% posterior predictive interval of Y")

Figure 5-4

ノイズが正規分布に従うと仮定していたので, 後者のノイズの分布確認は, ノイズの分布と正規分布の比較をすればよい. 本ではヒストグラムを描いていたが, ここではより手軽にQQ-plotを使う.

ms$noiseは.stan内のgenerated quantitiesとして生成されているとする(Exercise5-2). このms$noiseの最頻値を持つnoise_dfを作ってプロットする.

noise_df <- ms$noise %>%
    apply(2, function(x) {
        dens <- density(x)
        mode_x <- dens$x[which.max(dens$y)]
    }) %>%
    data.frame() %>%
    magrittr::set_colnames("err_mode") %>%
    mutate(A = d$A) # optional. used only in geom_qq

お手軽にプロットするならR base graphsが便利.

qqnorm(noise_df$err_mode)
qqline(noise_df$err_mode, col = "red")

カテゴリ変数Aでプロットを分けたいならggplotのgeom_qq_xxxx.

noise_df %>%
    ggplot(mapping = aes(sample = err_mode, color = factor(A))) +
    geom_qq_line() +
    geom_qq() +
    guides(color = guide_legend("A")) +
    labs(title = "Normal QQ-plot of MAP error")

Model 5-4 二項ロジスティック回帰

  • カテゴリ変数A: アルバイトが好きかどうか
  • 連続値変数Score: 学問への興味の強さ
  • 正整数変数M: 履修登録した科目の総授業回数 の3変数で
  • 応答変数Y: 実際に出席した回数 を二項ロジスティック回帰で説明する.
fit <- stan(file = stan_path, data = data, seed = 1234)
ms <- rstan::extract(fit)

Figure 5-10

予測された出席確率と実測値をプロットしてあてはまりの良さを確認する. ここでは, bayesplot::ppc_scatter_avg_dataで(実測値, 予測確率, カテゴリA)の情報を持ったデータフレームを吐かせ, それをもとにggplotでヴァイオリンプロットを描く.

# process data for plot
data_plot <- bayesplot::ppc_scatter_avg_data(y = d$Y, yrep = ms$q, group = d$A) %>% transmute(A = group, y_obs = factor(y_obs), prob_pred = value)
# plot
data_plot %>%
    ggplot(aes(x = prob_pred, y = y_obs)) +
    facet_wrap(~A, ncol = 2, scales = "free_x", labeller = label_both) +
    geom_violin(trim = FALSE, size = 1, color = "grey80") +
    geom_point(aes(color = y_obs), position = position_jitter(w = 0, h = 0.2), size = 1, alpha = 0.5) +
    guides(color = "none") +
    labs(x = "Predicted Probability", y = "Y Observed", title = "Violin plot of Observed Y vs Predicted Probability")

Model 5-6 ポアソン回帰

AやScoreの変動が応答変数Mに与える限界効果はbayesplot::mcmc_areaなどでプロットできる. この場合, stanファイルの中でexp(A)などのプロットしたい変数のMCMCサンプルを生成しておく必要がある.

mcmc_areas(fit, pars = c("A_MAE", "Score_MAE[10]", "Score_MAE[5]", "Score_MAE[1]"), prob = 0.90, prob_outer = 0.95) +
    scale_y_discrete(labels = c("A_MAE" = "A 0->1", "Score_MAE[1]" = "Score +10", "Score_MAE[5]" = "Score +50", "Score_MAE[10]" = "Score +100")) +
    grid_lines(color = "gray80") +
    labs(title = "95% Posterior Credible Intervals of Marginal Effect of Score and A factor", x = "Marginal Effect on Y (times)")
